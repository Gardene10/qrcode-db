Ao dev [para testar se está funcionando o cors da aplicação basta seguir os passos]

1.0 Substituir'http://localhost:5000/ping' pelo URL real da rota você irá usar

1.2 Arrasta o arquivo HTML no seu navegador

1.3 Clica ai no botão "Testar CORS DA APLICAÇÃO".

1.4 O JavaScript fará uma solicitação GET para a api em http://localhost:5000.

1.5 Verifica se no o console do navegador para ver a resposta da API ou qualquer erro de CORS.

exemplificando quando da ruim a resposta do console fica em vermelho e com a mensagem de bloqueio de cors como erro faltou o cabeçalho AccessControlAllowOrigin
